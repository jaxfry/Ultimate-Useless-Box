# Ultimate-Useless-Box

The Ultimate Useless Box, a box that does nothing, with style.

Heavily inspired by LuuMa EV3, I wanted my own useless box, until I realized that Lego's Mindstorm is prohibitively expensive. I decided to make my own using standard electronics components with a few unique twists of my own :)

## Project Goals
- Build a reliable useless box with some personality
- Use accessible parts and open files
- Learn more about PCB design
