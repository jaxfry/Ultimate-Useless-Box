# System Overview

Goal: flip the user switch back off in a fun way, every time.

## Main Pieces
- Input: user switch interaction
- Logic: microcontroller state machine
- Output: actuator for arm and lid motion
- Power: safe and stable supply

## Things I Care About
- Works reliably
- Not too noisy
- Easy to tweak and repair
